# 🛠️ Technology Stack - Ocean Infinity Website

## **React, HTML, CSS Implementation**

This project is built using **pure React, HTML, and CSS** - no additional frameworks or libraries beyond the core React ecosystem.

---

## **React Components (JavaScript)**

### **File Structure:**

```
src/
├── App.js              # Main application component
├── index.js            # React entry point
└── components/         # React functional components
    ├── Navbar.js       # Navigation component
    ├── Hero.js         # Hero section component
    ├── About.js        # About section component
    ├── Strengths.js    # Strength cards component
    ├── Products.js     # Product categories component
    ├── Certifications.js # Certifications component
    ├── Contact.js      # Contact form component
    └── Footer.js       # Footer component
```

### **React Features Used:**

- ✅ **Functional Components** - Modern React syntax
- ✅ **React Hooks** - useState, useEffect for state management
- ✅ **JSX** - HTML-like syntax in JavaScript
- ✅ **Component Composition** - Reusable component architecture
- ✅ **Event Handling** - Interactive user interactions
- ✅ **Conditional Rendering** - Dynamic content display

- ✅ **Maintenance** - Simple and straightforward codebase
